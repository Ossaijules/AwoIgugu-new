# Awo Igugu (Next.js)

This project has been migrated from Vite to Next.js (App Router).

## Run

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Production build:

```bash
npm run build
npm run start
```

## Routes

- `/`
- `/collection`
- `/Collection` (compatibility alias)
